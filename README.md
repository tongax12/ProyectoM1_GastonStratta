# 🎨 Generador de Paleta de Colores

Un generador interactivo de paletas de colores aleatorios con JavaScript puro. Genera colores en formato **HSL** y **HEX**, con opciones de bloqueo y copia rápida.

## ✨ Características

- 🎯 **Generación aleatoria** de colores en formato HSL y HEX
- 🔒 **Bloquea colores** para mantenerlos mientras generas nuevos
- 📋 **Copia rápida** del código HEX con un click
- 🌈 **Formas orgánicas** de colores (efecto derretido)
- 📱 **Responsive** - Funciona en cualquier dispositivo
- ⚡ **JavaScript puro** - Sin frameworks

## 🚀 Instrucciones de uso

1. **Selecciona cantidad de colores**: Elige entre 6, 8 o 9 colores
2. **Elige formato**: HSL o HEX
3. **Haz clic en "Generar Paleta"**: Los colores aparecerán en pantalla
4. **Bloquea colores** que te gusten con el botón 🔓 Bloquear
5. **Copia el código HEX** haciendo clic en el color

## 📋 Tecnologías usadas

- **HTML5** - Estructura
- **CSS3** - Estilos 
- **JavaScript puro** - Lógica 

## Pasos para descargar y ejecutar la aplicacion de forma local:

1- En GitHub, entrar al repositorio.
2- Una vez en el repositorio, hacer click en el botón de code y copiar el URL HTTPS que aparece.
3- Crear una carpeta para almacenar los archivos del repositorio.
4- Abrir la consola dentro de la carpeta o en Visual abrir la carpeta y abrir una terminal.
5- Usar el comando git clone y pegar el URL del repositorio copiado anteriormente.
6- Una vez que se descarguen todos los archivos, ir a la carpeta donde clonaste el repositorio y abrir el archivo HTML. Ahi se abrirá en tu navegador el proyecto.

## Pasos para desplegar la aplicación:

1- En GitHub, entrar al repositorio.
2- Hacer click en settings.
3- Bajar y buscar en la parte izquierda la opcion de Pages, hacer click.
4- En esa página buscamos en la parte de Build and deployment, buscar la parte de Branch.
5- Hacer click en None y seleccionar la opción de main y en el el nuevo menú elegir /(root).
6- Una vez seleccionado todo como se indicó, hacer click en el botón Save. Esperar 1 o 2 minutos y en la parte de arriba de la página va a aparecer el link despegable del proyecto.
